# HyperGit v0.1.0

发布日期：2026-05-31
里程碑：首个 native `hgx` MVP

## 本次发布包含

- 可运行的本地仓库主路径：`hgx init`、`status`、`add`、`commit`、`log`、`diff`。
- 工作区能力：`checkout`、`sparse`、`hydrate`、`dehydrate`、stage snapshot、local change tracking。
- 存储与协议能力：canonical codec v1、loose object store、segment pack、file remote `fetch/push`、HTTP remote smoke 路径。
- 性能与规模能力：parallel add/checkout/diff 基础设施、大文件 chunking、manifest query/diff 基准、Git 互操作原型。

## 发布结论

`v0.1.0` 标记了 HyperGit 第一版可运行的纯 Uya MVP。它对应设计文档里的 M0-M9 路线图，并完成 `docs/todo.md` 中的 Release Readiness 检查项。
